# 1. enum CPU_Rank（枚举类型）
---
### 1.1 枚举的作用

- `enum` 用于定义一组**有限且有意义的常量**  
- 让代码更可读，避免使用魔法数字  
- 限制变量取值范围，提高类型安全性  
- 在大型项目中，枚举常用于状态机、错误码、类型分类
---
### 1.2 值的自动递增规则

- `P1 = 1` 明确指定起始值为 1  
- 后续未指定的枚举项自动在前一个基础上 +1  

| 枚举名 | 实际整数值 |
|-------|------------|
| P1    | 1          |
| P2    | 2          |
| P3    | 3          |
| P4    | 4          |
| P5    | 5          |
| P6    | 6          |
| P7    | 7          |

`enum class`更安全，不会隐式转换为 int

---
### 1.3 使用方式

### 声明变量
```cpp
CPU_Rank r = P4;
```
### 整数转枚举（题目中用法）
```cpp
CPU_Rank r = (CPU_Rank)3;   // r = P3
```
`static_cast<CPU_Rank>(3)`，更安全

---
### 1.4 为什么要用枚举

- 代码更清晰  
- 避免使用难理解的数字  
- 便于维护和扩展  
- 在类中使用枚举可以让接口更规范  

例如：
```cpp
CPU cpu(P3, 2400, 11.5);
```
比：
```cpp
CPU cpu(3, 2400, 11.5);
```
更直观。
