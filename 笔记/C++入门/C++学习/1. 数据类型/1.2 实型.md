# 实型（浮点型）

---
## 1.1 浮点型的分类与范围

浮点型用于表示带小数的实数，分为==单精度==与==双精度==两类。

| 类型（type） | 中文名称   | 精度（有效位数）  | 字节数（bytes） | 取值范围（range）        |
| -------- | ------ | --------- | ---------- | ------------------ |
| `float`  | 单精度浮点数 | 约 7 位     | 4 字节       | -3.4E38 ~ 3.4E38   |
| `double` | 双精度浮点数 | 约 15~16 位 | 8 字节       | -1.7E308 ~ 1.7E308 |

---
## 1.2 小数位控制

`%.nf` —— 保留 **n 位小数**
---
- 
    1. `%.2f` → 保留 2 位小数  
    2. `%.3f` → 保留 3 位小数  
    3. `%.0f` → 不保留小数（默认四舍五入）

[[setprecision();]] —— 保留 **n 位小数**
---
---
## 1.3 示例（Examples）

```cpp
float num1 = 3.1415;
cout << num1 << endl;        // 3.1415
printf("%.2f", num1);        // 3.14
```
