# 转义字符（escape sequences）

---
## 1.1 转义字符的作用

转义字符用于表示无法直接输入的特殊字符，例如换行、制表符、引号等。
---
---
## 1.2 常见转义字符列表
- 
    1. `\a` —— 警报（响铃）  
    2. `\b` —— 退格  
    3. `\f` —— 换页  
    4. `\n` —— 换行符  
    5. `\r` —— 回车  
    6. `\t` —— 制表符（tab）  
    7. `\\` —— 反斜杠  
    8. `\'` —— 单引号  
    9. `\"` —— 双引号  
---
## 1.3 示例
```cpp
cout << "Hello\nWorld";  
// 输出：
// Hello
// World
```

```cpp
cout << "A\tB\tC";  
// 输出：A    B    C
```

```cpp
cout << "say: \"Hi\"";  
// 输出：say: "Hi"
```
