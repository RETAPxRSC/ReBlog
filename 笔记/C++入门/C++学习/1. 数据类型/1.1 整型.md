# 整型
---
## 1.1 整型的分类与范围

常见整型的字节数与取值范围：

| 类型（type）       | 中文名称  | 字节数（bytes） | 取值范围（range）      |
| -------------- | ----- | ---------- | ---------------- |
| `int`          | 整型    | 4 字节       | -2^31 ~ 2^31 - 1 |
| `short`        | 短整型   | 2 字节       | -2^15 ~ 2^15 - 1 |
| `long`         | 长整型   | 4 字节       | -2^31 ~ 2^31 - 1 |
| `long long`    | 长长整型  | 8 字节       | -2^63 ~ 2^63 - 1 |
| `unsigned int` | 无符号整型 | 4 字节       | 0 ~ 2^32 - 1     |

---
## 1.2 示例

```cpp
int a = 10;          // 普通整型
short b = 5;         // 短整型
long c = 100000;     // 长整型
long long d = 123456789; // 长长整型
unsigned int e = 20; // 无符号整型
```
