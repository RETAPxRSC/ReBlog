# sizeof 关键字（sizeof operator）

---
## 1.1 sizeof 的语法
`sizeof (变量名)`
---
`sizeof (数据类型)`
---
---
## 1.2 sizeof 的作用
- 
    1. 用于获取 **数据类型** 所占的字节数  
    2. 用于获取 **变量** 在内存中实际占用的字节数  
    3. 结果在编译期确定（不是运行时）  
    4. 返回值类型为 `size_t`（无符号整数类型）  
---
## 1.3 示例

```cpp
short num1 = 10;

cout << sizeof(num1) << endl;  // 2
cout << sizeof(short) << endl; // 2
```
