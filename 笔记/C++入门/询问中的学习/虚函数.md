# 1. 虚函数（virtual function）
---
### 1.1 定义

虚函数是使用关键字 `virtual` 修饰的成员函数，用于在继承体系中实现 **多态（polymorphism）**。

```cpp
virtual double GetArea();
```

---

### 1.2 虚函数的作用

- 允许 **基类指针 / 引用** 调用 **派生类自己的函数版本**
- 实现 **运行时多态（dynamic binding）**
- 让继承体系更灵活、更可扩展

例如：

```cpp
Shape* p = new Circle(10);
cout << p->GetArea();   // 调用 Circle 的 GetArea()
```

如果没有 `virtual`，上面这句会调用基类的版本，失去多态意义。

---

### 1.3 虚函数的语法结构

```cpp
virtual 返回类型 函数名(参数列表);
```

例如：

```cpp
virtual double GetArea();
```

---

### 1.4 纯虚函数（pure virtual function）

如果一个虚函数在基类中不需要实现，可以写成：

```cpp
virtual double GetArea() = 0;
```

含义：

- 这是一个 **纯虚函数**
- 派生类必须实现它
- 含有纯虚函数的类称为 **抽象类（abstract class）**
- 抽象类不能实例化

---

### 1.5 虚函数的调用机制（虚函数表 vtable）

（你未来一定会用到，这里适当补充）

- 每个含虚函数的类都有一个 **虚函数表（vtable）**
- 对象内部保存一个指向 vtable 的指针（vptr）
- 通过 vtable 实现 **运行时动态绑定**

---

### 1.6 override（派生类重写虚函数）

派生类重写虚函数时可以加上 `override`，更安全：

```cpp
double GetArea() override {
    return length * width;
}
```

---

### 1.7 虚函数与多态的核心关系

- **没有 virtual → 静态绑定 → 调用基类版本**
- **有 virtual → 动态绑定 → 调用派生类版本**

这是 C++ 面向对象的核心机制之一。
