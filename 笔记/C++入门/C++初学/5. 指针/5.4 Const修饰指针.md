# Const修饰指针
---
## 1.1 const 修饰指针（常量指针）
常量指针：`int *const p = &a;`
---
- 
    1. 指针本身是常量 → **指针的指向不可修改**  
    2. 指针指向的值可以修改  
---
## 1.2 const 修饰常量（指针常量）
指针常量：`const int *p = &a;`
---
- 
    1. 指针指向的值是常量 → **指针的值不可修改**  
    2. 指针的指向可以修改  
---
## 1.3 const 同时修饰指针与常量
`const int *const p = &a;`
---
- 
    1. 指针的指向不可修改  
    2. 指针指向的值不可修改  
---
```
 * 此为指针指向的值 <<<----vvv
 * p1(0x0011) >>> 0x0011(10) <<< [a]
 *            >>> 0x0022(10) <<< [b]
 * -----------^^^------>>> 此为指针的指向
```
---
## 1.4 三种 const 指针
### 1. 指针常量：`const int *p`
- 
    1. `p = &b;` ✅ 可以修改指向  
    2. `*p = 100;` ❌ 不可修改值  
### 2. 常量指针：`int *const p`
- 
    1. `p = &b;` ❌ 不可修改指向  
    2. `*p = 100;` ✅ 可以修改值  
### 3. 双重 const：`const int *const p`
- 
    1. `p = &b;` ❌ 不可修改指向  
    2. `*p = 100;` ❌ 不可修改值
