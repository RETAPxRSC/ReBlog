# 引用（reference）
---
## 1.1 引用的概念
引用：引用已存在变量的别名。
---
---
- 
    1. 引用本质上是变量的“别名”  
    2. 对引用的操作就是对原变量的操作  
---
## 1.2 引用的语法
`数据类型` `&别名` = `已存在变量名;`
---
---
## 1.3 引用的注意事项
- 
    1. 引用在定义时必须初始化  
    2. 引用一旦初始化，就不能改变引用的对象  
    3. 引用不是指针，不能为 null  
---
## 1.4 引用做函数参数
作用：让形参直接操作实参（类似指针传递，但语法更简洁）。
---
- 
    1. 值传递：形参是实参的拷贝  
    2. 地址传递：传入指针，操作实参  
    3. 引用传递：形参是实参的别名，操作实参  

---
## 1.5 引用做函数返回值
- 
    1. 引用可以作为函数返回值  
    2. ==必须保证==返回的引用指向的对象仍然存在  
    3. 不能返回局部变量的引用（生命周期结束）  
    4. 可以返回静态变量的引用  
---
## 1.6 示例代码：引用的完整演示
---
```cpp
// 引用

/*
 * 引用：引用已存在变量的别名
 *
 * 语法：
 *      数据类型 &别名 = 已存在变量名;
 *
 * 引用的注意事项：
 *      引用在定义时必须初始化
 *      引用一旦初始化，就不能改变引用的对象
 *
 * 引用做函数参数：
 *      作用：函数传参时，可以运用引用让形参直接操作实参
 * 
 * 引用做函数的返回值：
 *      作用：引用可以作为函数的返回值
 *      注意事项：引用做函数的返回值时，要注意返回值的生命周期
 */

#include <iostream>
using namespace std;

// 值传递
void func(int a, int b)
{
    int temp = a;
    a = b;
    b = temp;
    cout << "f1 a = " << a << endl;
    cout << "f1 b = " << b << endl;
    cout << endl;
}

// 地址传递
void func2(int *a, int *b)
{
    int temp = *a;
    *a = *b;
    *b = temp;
    cout << "f2 a = " << *a << endl;
    cout << "f2 b = " << *b << endl;
    cout << endl;
}

// 引用传递
void func3(int &a, int &b)
{
    int temp = a;
    a = b;
    b = temp;
    cout << "f3 a = " << a << endl;
    cout << "f3 b = " << b << endl;
    cout << endl;
}

int& test01()
{
    int alpha = 10; // 此时 alpha 是局部变量
    return alpha;
}

int& test02()
{
    static int alpha2 = 10; // alpha2 是静态局部变量
    return alpha2;
}

int main()
{
    int a = 10;
    int b = 20;
    
    int &c = a;
    // 引用必须初始化
    c = 20;
    // 引用的对象改变，原变量也会改变
    cout << "a = " << a << endl;
    cout << "c = " << c << endl;
    cout << endl;

    a = 10;
    b = 20;
    
    func(a, b);
    cout << "mf1 a = " << a << endl;
    cout << "mf1 b = " << b << endl;
    cout << endl;

    a = 10;
    b = 20;

    func2(&a, &b);
    cout << "mf2 a = " << a << endl;
    cout << "mf2 b = " << b << endl;
    cout << endl;

    a = 10;
    b = 20;

    func3(a, b);
    cout << "mf3 a = " << a << endl;
    cout << "mf3 b = " << b << endl;
    cout << endl;

    int &ref = test01();
    cout << "ref = " << ref << endl; // ref 引用的是 alpha
    cout << "alpha = " << alpha << endl;
    cout << "ref = " << ref << endl; // 此时已经销毁
    cout << endl;

    int &ref2 = test02();
    cout << "ref2 = " << ref2 << endl; // ref2 引用的是 alpha2
    cout << "alpha2 = " << alpha2 << endl;
    cout << "ref2 = " << ref2 << endl;
    cout << endl;

    return 0;
}
```
---
## 1.7 小结
- 
	1. 引用是变量的别名
        
    2. 引用必须初始化且不能更改指向
        
    3. 引用传参比指针更简洁
        
    4. 返回引用时必须保证对象仍然存在
