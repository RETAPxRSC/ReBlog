# 常量（constant）
---
## 1.1 常量的定义方式

### 宏常量（macro constant）

`#define (常量名) (常量值)`
---
- 
	1. **宏常量**在预处理阶段展开  
	2. **不占用内存**（仅文本替换）  
	3. **没有类型检查**，调试困难  
---
### const 常量

`const (数据类型) (变量名)` = `初始值` ;
---
- 
    1. `const` 修饰的变量 **具有类型**  
    2. 编译器会进行 **类型检查**  
    3. 变量 **不可修改**  
    4. 比 `#define` 更安全 
---
## 1.2 示例
```cpp
#define Day 7   // 一周有7天

int month1 = 12;  // 一年有12个月
month1 = 13;      // 可以修改
const int month2 = 12; // 一年有12个月
month2 = 13;           // 不能修改（编译报错）
```
