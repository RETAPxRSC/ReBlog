# 指针与函数（pointer & function）
---
## 1.1 利用指针作为函数参数
利用指针做函数参数，可以修改实参的值（地址传递）。
---
- 
    1. 值传递：形参修改==不影响实参==  
    2. 地址传递（指针传递）：形参指向实参 → ==可以==修改实参  
---
## 1.2 地址传递示例：swap 函数
---
```cpp
void swap(int *a, int *b) // 传入 a 和 b 的地址
{
    int temp = *a;
    *a = *b;
    *b = temp;
}

int main()
{
    int a = 10;
    int b = 20;

    swap(&a, &b); // 将 a 和 b 的地址作为参数传递，交换 a 和 b 的值

    cout << a << endl;
    cout << b << endl;

    return 0;
}
```
---
## 1.3 地址传递的特点
- 
    1. 传入的是变量的地址  
    2. 函数内部通过 `*指针` 修改的是实参本身  
    3. **地址传递可以修饰实参**（与值传递不同）
