# 变量（variable）

---
## 1.1变量的语法
`(数据类型)` `(变量名)` = `初始值` ;
---
---
## 1.2变量的准则
- 
	 1. `变量名` ==只能==包含字母、数字和下划线
	 2. `变量名` ==不能==以数字开头
	 3. `变量名` 区分大小写
	 4. `变量名` ==不能==是C++关键字
---
## 1.3变量的类型
- 
	1. `int` 整数类型
	2. `float` 单精度浮点数类型
	3. `double` ==双精度==浮点数类型
	4. `char` 字符类型
	5. `bool` 布尔类型
---
```cpp
	char ch = 'a'; // 字符
	char str[] = "hello"; // 字符串
```
