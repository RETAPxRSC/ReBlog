# 跳转语句

---
## 1.1 break 语句

`break` 用于**终止当前结构**，常见于 switch 与循环中。
---
---
- 
    1. 出现在 `switch` 中：终止当前 `case`，跳出整个 `switch`  
    2. 出现在循环中：跳出当前循环，执行循环后面的语句  
    3. 出现在嵌套循环中：只跳出**当前层**循环，不影响外层循环  
### 基本语法

```cpp
for (int i = 0; i < 5; i++) {
    if (i == 3) break;
    cout << i << endl;
}
// 输出：0 1 2
```
---
## 1.2 continue 语句

`continue` 用于**跳过本次循环剩余语句**，直接进入下一次循环。
---
### 基本语法

```cpp
for (int i = 0; i < 5; i++) {
    if (i == 2) continue;
    cout << i << endl;
}
// 输出：0 1 3 4
```
---
## 1.3 goto 语句

`goto` 用于跳转到指定标签位置，执行标签后的语句。  
---
---
### 基本语法

```cpp
cout << "A" << endl;
goto END;

cout << "B" << endl; // 不执行

END:
cout << "C" << endl;
// 输出：A C
```
