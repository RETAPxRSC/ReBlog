# 结构体数组（struct array）
---
## 1.1 结构体数组的概念
结构体数组：将自定义的结构体放入数组中，方便批量管理与维护。
---
- 
    1. 本质是“数组 + 结构体”的组合  
    2. 每个元素都是一个结构体变量  
    3. 常用于管理一组相同类型的数据（如学生列表、员工列表）  

---
## 1.2 创建结构体数组
结构体数组的基本创建方式如下：
---
```cpp
struct 结构体名 数组名[元素个数] =
{
    {成员1值, 成员2值, ...},
    {成员1值, 成员2值, ...},
    ...
};
```
---
## 1.3 给结构体数组赋值
---
```cpp
struct Student StuArr[3] =
{
    {"Name1", Age1, Score1},
    {"Name2", Age2, Score2},
    {"Name3", Age3, Score3},
};
```
补充说明：  
- 初始化时必须保证成员顺序一致  
- 字符串成员可以直接用字符串字面量初始化  
- 数组大小可以省略：`Student StuArr[] = { ... };`
---
## 1.4 遍历结构体数组
---
```cpp
for (int i = 0; i < 3; i++)
{
    cout << "姓名：" << StuArr[i].name << endl;
    cout << "年龄：" << StuArr[i].age << endl;
    cout << "成绩：" << StuArr[i].score << endl;
}
```
补充说明：  
- 使用 `StuArr[i].成员名` 访问结构体成员  
- 若使用指针遍历，可写成：`(StuArr + i)->name`  
---
## 1.5 常见补充
- 
    1. 结构体数组可以与函数配合使用（传入整个数组或传入单个元素）  
    2. 结构体数组可以排序（按成绩、年龄等）  
    3. 结构体数组元素之间可以直接赋值（整体拷贝）
