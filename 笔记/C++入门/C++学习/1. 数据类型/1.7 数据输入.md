## 1.1 cin 输入规则
- 
    1. 空格、Tab、换行符 都可以作为 **分隔符**  
    2. 可以一次输入多个数据，用空格、Tab、换行符分隔  
    3. 输入字符串时，遇到空格停止 
    4. `cin >> 变量名;` 基本输入格式  
	 spe：[[cin.ignore();]]

## 1.2 按数据类型分类的输入示例

---
### [[1.1 整型]]（int）

```cpp
int a;
cin >> a;
cout << a << endl;
```

---
### [[1.2 实型]]（double）
