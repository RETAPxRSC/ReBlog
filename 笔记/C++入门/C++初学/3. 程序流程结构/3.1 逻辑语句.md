# 逻辑语句（logic statements）

## 1.1 if 语句（if statement）

```cpp
if (条件) {
    语句;
}
```
## 1.2 if...else 语句

```cpp
if (条件) {
    语句1;
} else {
    语句2;
}
```
## 1.3 if...else if...else 语句

```cpp
if (条件1) {
    语句1;
} else if (条件2) {
    语句2;
} else {
    语句3;
}
```
## 1.4 switch 语句（switch-case）

```cpp
switch (表达式) {
    case 值1:
        语句1;
        break;
    case 值2:
        语句2;
        break;
    default:
        默认语句;
        break;
}
```
## 1.5 while 循环（while loop）

```cpp
while (条件) {
    循环体;
}
```
## 1.6 do...while 循环（do-while loop）

```cpp
do {
    循环体;
} while (条件);
```
## 1.7 for 循环（for loop）

```cpp
for (初始化; 条件; 更新) {
    循环体;
}
```
