# 关键字（keywords）

---
## 1.1 关键字定义

C++ 中的关键字（keyword）是保留字，具有特殊含义，不能作为变量名、函数名或标识符使用。

---

    1. `asm` —— 汇编指令  
    2. `auto` —— 自动类型推导  
    3. `bool` —— 布尔类型  
    4. `break` —— 跳出循环  
    5. `case` —— 开关语句

    7. `char` —— 字符类型  
    8. `class` —— 类  
    9. `const` —— 常量  
    10. `continue` —— 继续循环  
    11. `default` —— 默认  
    12. `delete` —— 删除  
    13. `do` —— 循环  
    14. `double` —— 双精度浮点数  
    15. `else` —— 否则  
    16. `enum` —— 枚举  
    17. `extern` —— 外部  
    18. `float` —— 单精度浮点数  
    19. `for` —— 循环  
    20. `friend` —— 友元  
    21. `goto` —— 跳转  
    22. `if` —— 如果  
    23. `inline` —— 内联  
    24. `int` —— 整数  
    25. `long` —— 长整数  
    26. `mutable` —— 可变

    28. `new` —— 新建对象  
    29. `operator` —— 运算符

    32. `public` —— 公共

    34. `return` —— 返回

    40. `switch` —— 开关

    43. `throw` —— 抛出异常  
    44. `true` —— 真  
    45. `try` —— 尝试

    53. `volatile` —— 易变

    55. `while` —— 循环

## 1.3 示例

```cpp
int a = 10;     // 使用 int 关键字定义整数
const int b = 5; // 使用 const 定义常量
class Person {}; // 使用 class 定义类
```
