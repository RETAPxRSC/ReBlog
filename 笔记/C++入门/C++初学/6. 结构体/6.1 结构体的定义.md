# 结构体的定义与使用（struct）
---
## 1.1 结构体的概念
结构体：用户自定义的数据类型，允许存储**不同类型**的数据。
---
---
## 1.2 结构体的语法
结构体的基本定义格式：
---
```cpp
struct 结构体名
{
    结构体成员列表
}; // ✅ 注意：结构体定义后必须加分号
```
---
## 1.3 创建结构体变量的方式
- 
    1. `struct 结构体名 变量名;`  
    2. `struct 结构体名 变量名 = {成员1值, 成员2值, ...};`  
    3. ✅ C++ 中 `struct` 可以省略：`结构体名 变量名;`
---
## 1.4 定义结构体时创建变量
---
```cpp
struct Student
{
    string name; // 姓名
    int age;     // 年龄
    int score;   // 成绩
} // ← 这里可以直接创建变量
```
---
## 1.5 示例：结构体变量的创建方式
---
### 方式 1：先定义结构体，再创建变量
```cpp
struct Student Stu1;
{
    "Name";
    Age;
    Score;
}
```
### 方式 2：定义时直接初始化
```cpp
struct Student Stu1 = {"Name", Age, Score};
```
