# 字符串（string）
---
## 1.1 字符串的两种风格

C++ 中有两种常见的字符串表示方式：**C 风格字符串** 与 **C++ 风格字符串**。

---
## 1.2 C 风格字符串（Cstring）
- 
    1. 使用 `char` 数组存储  
    2. 以 **`\0` 作为结尾标志**  
    3. 本质为字符数组  
---
## 1.3 C++ 风格字符串（String）
- 
    1. 使用 `string` 类（需要 `#include <string>`）  
    2. 更安全、功能更强  
    3. 支持长度获取、拼接、查找等操作
