# 内存分区（memory layout）

---
## 1.1 四大区域概念
- 
    1. **代码区**  
       - 存放函数体的二进制机器指令  
       - 由操作系统管理  
       - 只读  
       - 可共享  

- 
    2. **全局区**  
       - 存放：全局变量、静态变量、常量  
       - 包含：常量区（字符串常量池、其他常量）  
       - 程序结束后由操作系统释放  

- 
    3. **栈区**  
       - 编译器自动分配与释放  
       - 存放：函数参数、局部变量  
       - 函数结束后自动释放  
       - ⚠ 不要返回栈区地址  

- 
    4. **堆区**  
       - 程序员手动分配与释放  
       - 若未释放，程序结束时由系统回收  
       - Cpp 中主要使用 `new` 开辟内存  
---
## 1.2 new 操作符
- 
    1. `new` 在堆区开辟内存  
    2. 返回开辟空间的地址  
    3. 需要使用 `delete` 手动释放  
---
## 1.3 各区域地址
---
```cpp
// 代码区

/*
 * 内存分区模型：
 * 四大区域：
 * 1.代码区 ： 存放函数体的二进制代码，由操作系统进行管理的
 * 2.全局区 ： 存放全局变量、静态变量、常量 
 * 3.栈区 ： 由编译器自动分配释放，存放函数的参数值、局部变量等
 * 4.堆区 ： 由程序员分配和释放，若程序员不释放，程序结束时由操作系统回收
 *
 * 1.1 代码运行前：
 * 编译后 >> EXE文件 >> 未执行时：
 * 代码区：
 *      存放机器指令
 *      代码区是共享的
 *      代码区是只读的
 * 
 * 全局区：
 *      存放全局变量、静态变量
 *      全局区还包含了常量区，常量字符串池、其他常量
 *      全局区的数据在程序结束后由操作系统释放
 *
 * 栈区：
 *      由编译器自动分配释放，存放函数的参数值，局部变量等；
 *      栈区的数据在函数调用结束后自动释放，不要返回栈区地址
 * 
 * 堆区：
 *      由程序员分配和释放，若程序员不释放，程序结束时由操作系统回收
 *      在C++中，主要利用new在堆区开辟内存
 * 
 * >> new操作符：   
 *      利用new操作符在堆区开辟数据
 *      堆区开辟的数据，由程序员手动释放(delete)，若不释放，程序结束时由操作系统回收
 * 
 */

#include <iostream>
using namespace std;

int g_a = 10;
int g_b = 20;
const int gc_a = 10;
const int gc_b = 20;

int* func()
{
    int *a = new int(10);
    return a;
}

int main()
{
    cout << "全局区" << endl;
    cout << "\t全局变量g_a = " << (long long)&g_a << endl;
    cout << "\t全局变量g_b = " << (long long)&g_b << endl;
    cout << "\t全局常量gc_a = " << (long long)&gc_a << endl;
    cout << "\t全局常量gc_b = " << (long long)&gc_b << endl;

    cout << endl;

    cout << "栈区" << endl;
    int a = 10;
    int b = 20;
    cout << "\t局部变量a = " << (long long)&a << endl;
    cout << "\t局部变量b = " << (long long)&b << endl;

    cout << endl;

    cout << "堆区" << endl;
    int *p = func();
    cout << "\t堆区变量p = " << (long long)p << endl;
    cout << "\t堆区变量p指向的值 = " << *p << endl;
    delete p;

    cout << endl;

    cout << "静态区" << endl;
    static int s_a = 10;
    static int s_b = 20;
    cout << "\t静态变量s_a = " << (long long)&s_a << endl;
    cout << "\t静态变量s_b = " << (long long)&s_b << endl;

    cout << endl;

    cout << "字符串常量 = " << (long long)"hello world" << endl;

    cout << endl;

    cout << "局部常量" << endl;
    const int c_a = 10;
    const int c_b = 20;
    cout << "\t局部常量c_a = " << (long long)&c_a << endl;
    cout << "\t局部常量c_b = " << (long long)&c_b << endl;
}
```
---
## 1.4 小结
- 
	1. C++ 程序运行时内存分为四大区域
        
    2. 不同区域的生命周期不同
        
    3. `new` / `delete` 是堆区管理的核心
        
    4. 栈区变量不要返回地址
        
    5. 全局区与常量区由系统统一管理
