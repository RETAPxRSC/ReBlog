# 字符型（char type）
---
## 1.1 字符型的基本信息

`char` 用于存储单个字符，占 **1 字节**，范围为 **-128 ~ 127**。
---
---
## 1.2 字符型的特点
- 
    1. `char` 变量只能存储 **一个字符**  
    2. 字符本质上存储的是 **ASCII 码值**  
    3. 可以通过强制类型转换 `(int)` 查看字符对应的 ASCII 值  
    4. 字符串由多个字符组成，以 `'\0'` 结尾  
---
## 1.3 示例

```cpp
char ch = 'a';  // 字符型变量只能存储一个字符
cout << ch << endl;      // a
cout << (int)ch << endl; // 97 (ASCII 码值)
```
---
## Extra 字符串相关

```cpp
char str[] = "hello"; // 字符串（以 '\0' 结尾）
cout << str << endl;  // hello
```
