# 函数（function）
---
## 1.1 函数的作用
- 
    1. 封装一段特定的代码，减少重复代码  
    2. 将一个较大的程序分割为若干函数模块，每个模块完成一个特定功能  
---
## 1.2 函数的定义格式

函数组成：
---
- 
    1. 返回值类型  
    2. 函数名  
    3. 参数列表  
    4. 函数体语句  
    5. return 表达式  
### 基本结构：

```cpp
返回值类型 函数名(参数列表)
{
    函数体语句;
    return 表达式;
}
```
---
## 1.3 示例：加法函数

拆解：
---
- 
    1. 返回值类型：`int`  
    2. 函数名：`add`  
    3. 参数列表：`(int num1, int num2)`  
    4. 函数体语句：`int sum = num1 + num2;`  
    5. return 表达式：`return sum;`  
```cpp
int add(int num1, int num2)
{
    int sum = num1 + num2;
    return sum;
}
```
---
## 1.4 实参与形参
- 
    1. **实参**：调用函数时传递给函数的具体数值  
    2. **形参**：函数定义中声明的变量，用于接收实参  
---
## 1.5 值传递
- 
    1. 调用函数时，将实参的值复制给形参  
    2. 函数内部对形参的修改不会影响实参  
    3. 可以在函数内部使用形参进行计算，而不会改变实参  
---
## 1.6 四种函数类型

### 分类：
- 
    1. `void t01()` —— 无参无返回值  
    2. `void t02(int a)` —— 有参无返回值  
    3. `int t03()` —— 无参有返回值  
    4. `int t04(int a)` —— 有参有返回值  
### 结构：

```cpp
void t01() {}

void t02(int a) {}

int t03()
{
    return 0;
}

int t04(int a)
{
    return a;
}
```
---
## 1.7 函数的声明
- 
    1. 告诉编译器函数的存在（包含函数名、返回值类型、参数列表）  
    2. 声明可在调用之前或之后出现  
    3. 声明可多次出现，但定义只能出现一次  
---
## 1.8 函数的分文件编写
- 
    1. 创建 `.h` 头文件  
    2. 创建 `.cpp` 源文件  
    3. 在 `.h` 中声明函数  
    4. 在 `.cpp` 中定义函数  
    5. 在主文件中 `#include` 头文件  
---
## 1.9 构造函数的应用场景
- 
	1. 初始化成员变量
	    - 为结构体或类的成员变量赋初始值，避免对象处于未定义状态。
	2. 动态控制初始化
	    - 可以根据参数灵活地初始化不同状态的对象。
	3. 封装复杂逻辑
	    - 在构造函数中封装一些初始化逻辑，简化外部代码。
	4. 提高代码安全性
	    - 使用委托构造或删除某些构造函数可以避免意外使用。
---
## 总结
- 
	1. 构造函数是结构体中用来初始化成员变量的核心工具。
	2. 通过**默认构造**、**参数化构造**、**初始化列表**等形式，可以满足多样化的初始化需求。
	3. 推荐使用 **初始化列表**，特别是在初始化复杂类型（如 `std::vector`、引用、`const` 变量）时，效率更高。
