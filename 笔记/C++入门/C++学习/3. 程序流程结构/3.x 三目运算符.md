# 三目运算符（ternary operator）

---
## 1.1 基本语法

三目运算符用于根据条件表达式的真假，选择不同的结果。
---

语法格式：
`条件表达式 ? 表达式1 : 表达式2`
---
---
## 1.2 执行逻辑
- 
    1. 若 **条件表达式为真**，执行 **表达式1**，并返回表达式1 的结果  
    2. 若 **条件表达式为假**，执行 **表达式2**，并返回表达式2 的结果  
---
## 1.3 示例（Examples）

```cpp
int a = 10;
int b = 20;

int c = (a > b ? a : b);

cout << c << endl; // 20
```
