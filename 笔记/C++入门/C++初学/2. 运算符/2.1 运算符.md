# 运算符（operators）
---
## 1.1 算数运算符（arithmetic operators）

算数运算符用于执行基本数学运算。
---
---
### 1.1.1 加减乘除
- 
    1. 加法运算符：`+`  
    2. 减法运算符：`-`  
    3. 乘法运算符：`*`  
    4. 除法运算符：`/`  
---
### 1.1.2 取模运算符
- 
    5. 取模运算符：`%`（用于整数取余）
---
### 1.1.3 递增递减运算符
- 
    6. 递增运算符：`++`  
    7. 递减运算符：`--`  
前缀形式：`++a`、`--a` —— **先递增/递减，再使用**  
后缀形式：`a++`、`a--` —— **先使用，再递增/递减**

---
### 示例（Examples）

```cpp
int a = 5;
cout << a + 2 << endl; // 7
cout << a % 2 << endl; // 1

int b = 3;
cout << ++b << endl; // 4（先加再用）
cout << b++ << endl; // 4（先用再加）
```
---
## 1.2 赋值运算符（assignment operators）

赋值运算符用于给变量赋值或进行复合赋值。
---
---
- 
    1. 赋值：  ` = ` 
    2. 加等于：`+=`
    3. 减等于：`-=`
    4. 乘等于：`*=`
    5. 除等于：`/=`
    6. 模等于：`%=`
---
### 示例（Examples）

```cpp
int a = 10;
a += 5;  // 15
a *= 2;  // 30
```

---
## 1.3 比较运算符（comparison operators）

比较运算符用于比较两个值，结果为布尔值（true / false）。
---
---
- 
    1. 大于：`>`  
    2. 小于：`<`  
    3. 大于等于：`>=`  
    4. 小于等于：`<=`  
    5. 等于：` == `  
    6. 不等于：`!=`  
---
### 示例（Examples）

```cpp
cout << (4 > 3) << endl;  // 1
cout << (4 < 3) << endl;  // 0
cout << (4 == 3) << endl; // 0
cout << (4 != 3) << endl; // 1
```

---
## 1.4 逻辑运算符（logical operators）

逻辑运算符用于布尔逻辑判断。
---
---
- 
    1. 逻辑与：`&&` —— 全真为真，有假则假  
    2. 逻辑或：`||` —— 有真则真，全假为假  
    3. 逻辑非：`!`  —— 真变假，假变真  
---
### 示例（Examples）

```cpp
bool a = false;
bool b = true;

cout << !a << endl;       // 1
cout << (a && b) << endl; // 0
cout << (a || b) << endl; // 1
```
