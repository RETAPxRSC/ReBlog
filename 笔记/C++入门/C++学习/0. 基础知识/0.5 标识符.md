# 标识符命名规则（identifier）

---
## 1.1 标识符基本规则

标识符用于命名变量、函数、类等程序元素，必须遵守以下规范：
- 
    1. 标识符只能包含 **字母、数字、下划线**  
    2. 标识符 **不能以数字开头**  
    3. 标识符 **区分大小写**  
    4. 标识符 **不能是 C++ 关键字**  
---
## 1.2 命名建议：见名知意原则

==**“见名知意”**== ——标识符名称应能体现其含义，使代码更易读、更易维护。
- 
    1. 使用有意义的英文单词，如 `count`、`total`、`index`  
    2. 避免使用无意义缩写，如 `a1`、`xx`  
    3. 多词命名可使用下划线或驼峰命名，如 `student_name` 或 `studentName`  
---
## 1.3 示例

```cpp
int student_age;     // ✅ 见名知意
int a;               // ❌ 不推荐，含义不明确

int totalScore = 95; // ✅ 多词命名
int ts = 95;         // ❌ 不推荐
```
